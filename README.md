# indexofficial-hth-resource

一个用于归档和发布独立 HTML 页面的资源仓库。本仓库不针对任何特定域名或网站，仅用于存放、维护和共享若干静态 HTML 文件及其相关资源。

## 项目简介

本项目旨在提供一个简洁、稳定的 HTML 页面归档空间。所收录的页面均为独立功能或展示用途，不涉及商业推广、用户追踪或动态后端服务。仓库内容可被直接用于网页预览、资源引用或学习参考。

## 目录结构

```
.
├── pages/          # 存放各独立 HTML 页面及其附属资源
├── assets/         # 公共资源文件（如样式、脚本、图片等）
├── README.md       # 本说明文件
└── LICENSE         # 开源许可证
```

- `pages/` 下每个子目录对应一个独立的 HTML 页面，包含该页面所需的所有文件。
- `assets/` 用于存放多个页面可能共用的资源，减少重复。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖外部服务或数据库。
- 每个页面目录内通常包含 `index.html` 及必要的 CSS、JavaScript、图片等文件。
- 页面内容可能涉及信息展示、工具、实验性功能等，不保证持续更新或兼容所有浏览器。
- 归档的页面可能来自不同时间或场景，风格与用途各异。

## 使用方式

您可以直接通过 GitHub Pages 或本地服务器访问 `pages/` 下的 HTML 文件。例如：

```
https://<你的用户名>.github.io/indexofficial-hth-resource/pages/某个页面/
```

也可以将仓库克隆到本地，在浏览器中打开对应 HTML 文件。
